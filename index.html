Proyectos

Estos son algunos de los proyectos que he recientemente. Desde aplicaciones web hasta experimentos en 3D.

Ng Countries

Una aplicación de página única (SPA) desarrollada en Angular 15 que se conecta a la API RestCountries para mostrar información detallada sobre distintos países. La aplicación ofrece funcionalidades como búsqueda y filtrado por región, así como detalles específicos de cada país, como su capital, población, área, y más.

ThreeJS 3D Text Render

Un visor de texto tridimensional desarrollado con Three.js, una biblioteca JavaScript para la creación de gráficos en 3D en el navegador. Este proyecto muestra cómo renderizar texto en un entorno 3D utilizando Three.js, permitiendo al usuario interactuar con el texto y explorar diferentes perspectivas.
